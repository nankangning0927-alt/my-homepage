# 新增作品卡片

帮用户在 `index.html` 的作品区(`<section id="work">` 里的 `<div class="grid">`)
追加一张新的项目卡片。

## 步骤

1. 询问(或从用户的话里提取)这三样信息:
   - **项目名称**(卡片标题)
   - **项目简介**(一两句话)
   - **标签**(比如 "网页设计""产品设计""个人项目")
   - 如果用户没说全,用合理的占位内容补上,并提醒用户之后替换。

2. 在 `<div class="grid">` 内、最后一个 `</article>` 之后,按下面的模板追加卡片,
   保持与现有卡片完全一致的缩进和结构:

   ```html
   <article class="card">
     <h3>项目名称</h3>
     <p>项目简介。</p>
     <span class="tag">标签</span>
   </article>
   ```

3. 用 `git diff` 把改动展示给用户确认。

4. 用户确认后:`git add` → `git commit`(写清楚的中文提交信息)→
   `git push` 到当前工作分支。如果需要更新线上网页,再合并到 `main`。

## 注意

- 只在作品区追加,**不要改动**页面其他部分(导航、首页、联系方式、样式)。
- 卡片的 class、标签结构必须和现有卡片一致,保证样式正常。
